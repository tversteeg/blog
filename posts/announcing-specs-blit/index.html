Specs-blit is a library I started developing.

It's part of a set of tools that can be used to create game engines.

The library exposes the blit functionality in specs. It allows you to create Sprite components that can be positioned and rendered to a PixelBuffer. The sprites are all put on the heap and passed around as references.
